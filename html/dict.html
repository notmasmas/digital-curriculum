    • Tag Utilização
    <nav> Representa uma seção da página dedicada a providenciar links de navegação para seções do mesmo documento ou outras páginas.
    <menu> Alternativa semântica para uma lista não-ordenada. Guarda, em itens, links para seções do documento ou outras páginas
    <ul> Tag para uma lista genérica não-ordenada. Guarda itens de lista
    <li> Em inglês, "list item". Representa um item de uma lista
    <a> Cria um hyperlink para outras páginas da web, arquivos, endereços de e-mail e seções dentro da mesma página.
    <aside> Representa uma parte do conteúdo que não é a parte principal da página. Neste site, é utilizado como uma "side-bar"
    <main> Representa o conteúdo principal da página
    <section> Representa uma seção genérica do documento. Segundo as recomendações, elementos section sempre devem ter um heading (h1, h2, h3...), com poucas exceções
    <p> Representa um parágrafo de texto
    <h1>, <h2>...<h6> Representa headers de seções (cabeçalhos). Possuem um nível de hierarquia, onde h1 é de maior hierarquia, e h6 o de menor. É recomendado evitar utilizar múltiplos headers h1 em uma só página, mas há exceções.
    <footer> É o rodapé da página. Geralmente contém informaçes sobre o autor da página, créditos e/ou links úteis para o usuário
    <div> É uma divisória genérica, geralmente utilizada pra auxiliar no posicionamento de elementos no CSS
